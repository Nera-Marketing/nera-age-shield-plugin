DCMS Voluntary Code (Clause 1.1) age verification — collects and verifies date of birth and blocks anyone under 18 from entering prize draws.

== Description ==

**Nera – DCMS Age Gate** implements age verification for WooCommerce competition sites:

* Dialog-based date-of-birth collection and verification
* Blocks logged-in users under 18 from adding lottery tickets to the cart
* Registration and checkout DOB field integration
* Sitewide 18+ badge
* Server-side enforcement as a safety net for direct AJAX requests

**Requires** WooCommerce.

== Installation ==

1. Install and activate **WooCommerce**.
2. Upload and activate **Nera – DCMS Age Gate**.
3. Users will be prompted to verify their date of birth before participating in prize draws.
